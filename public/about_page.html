About Our Project

Our project provides real-time financial data for small investors and financial analysts, leveraging MarketStack and Fixer.io APIs for stock prices and currency exchange rates.

Data Sources

We use MarketStack for stock data and Fixer.io for currency exchange rates to ensure accurate and real-time updates.

Technology Stack

  • Front-end: React.js, D3.js
  • Back-end: Node.js, MongoDB
